🚂 Before We Start

What is Railway?

Railway is like a home for your website on the internet. Right now your app only works on YOUR computer. Railway makes it work for everyone, 24 hours a day!

YOU NEED THESE 4 FREE THINGS:

💻
Git
A tool that saves your code history
🐙
GitHub Account
A website that stores your code online
🚂
Railway Account
The service that runs your website
🔑
Your API Key
From Anthropic — you already have this!

Step 1 of 8

💻 Install Git

1 Open your browser (Chrome, Edge, or Safari)
2 Go to: git-scm.com/downloads
3 Click the big Download button for your computer
4 Run the installer — just keep clicking Next → Next → Finish. No changes needed!
✅ CHECK IT WORKED — open Terminal (or Command Prompt) and type:
git --version
# If you see "git version 2.xx.x" — it works! 🎉

Step 2 of 8

🐙 Create Your Free Accounts

1
GitHub (stores your code online)
Go to github.com → click "Sign up" → fill in name, email, password
2
Railway (runs your website)
Go to railway.app → click "Login" → choose "Login with GitHub"

Step 3 of 8

🖥️ Open Terminal

🪟
Windows
Press ⊞ Win key → type cmd → press Enter
🍎
Mac
Press ⌘ + Space → type terminal → press Enter

A black window appears. Now go to your project folder by typing:

Windows — type this (change "ASUS" to your username):
cd C:\Users\ASUS\claude\ielts-lms
Mac — type this:
cd ~/claude/ielts-lms
# Press Enter after typing. Nothing bad can happen!

Step 4 of 8

📦 Save Your Code With Git

Type these 3 commands one by one. Press Enter after each one and wait for it to finish.

# Command 1 — start tracking this folder
git init
# Command 2 — add ALL files to be saved
git add .
# Command 3 — save them with a message
git commit -m "first upload"
💡 Think of git commit like pressing Save on a document — but it remembers every save forever! You can always go back in time.
6 / 11
Step 5 of 8

☁️ Upload to GitHub

1 Go to github.com → click the + button (top-right) → click "New repository"
2 Name it ielts-lms → leave everything else as default → click "Create repository"
3 GitHub shows you commands — go back to Terminal and type them one by one:
git remote add origin https://github.com/YOUR-NAME/ielts-lms.git
git branch -M main
git push -u origin main
# Replace YOUR-NAME with your GitHub username!
🔑 If it asks for a password → use a Personal Access Token. In GitHub: Settings → Developer Settings → Personal Access Tokens → Generate new token (classic)
7 / 11
Step 6 of 8

🚂 Deploy on Railway!

1 Go to railway.app and log in
2 Click "New Project"
3 Click "Deploy from GitHub repo"
4 Find your ielts-lms repo and click it
5 Click "Deploy Now" — Railway builds your app automatically! ⏳ Wait 2–3 minutes...
🎉 You'll see a build log with lots of text scrolling — that's normal! When it says "Deployed" in green, you're done with this step.
8 / 11
Step 7 of 8

🔑 Add Your Secret Keys

In Railway: click your project → click the "Variables" tab → click "New Variable" and add these 4 things:

# Name (left box) Value (right box)
ANTHROPIC_API_KEY → sk-ant-xxxx (your Anthropic key)
JWT_SECRET → any-long-random-words-here
STARTING_BALANCE → 4.98
PORT → 3000
🔐 These are like passwords for your app. Railway keeps them safe and hidden. Never put your API key directly in your code!
9 / 11
Step 8 of 8

🌏 Get Your Live Website URL!

1 In Railway → click your project → click the "Settings" tab
2 Scroll down to "Domains" → click "Generate Domain"
3 Railway gives you a URL — it looks like this:
https://ielts-lms-production.up.railway.app

Copy it and share it with anyone! They can open it on any phone, tablet, or computer from anywhere in the world 📱💻🌍
